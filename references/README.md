# Style references

Reference images for the Berlin pixel-art style live here.

Plan for the MVP:

- 2–3 wide-angle "look and feel" references that match the late-90s
  SimCity / Rollercoaster Tycoon vibe of the NYC project.
- A small set of Berlin-specific reference renders: Wilhelminian
  Altbau roof shapes, Plattenbau facades, post-1990 government
  architecture (glass + sandstone), Spree water tiles, dense
  Tiergarten tree tiles.

Do not commit copyrighted screenshots from commercial games. Use
self-rendered references or clearly licensed sources only.
